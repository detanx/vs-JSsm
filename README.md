## vscode插件
1. 安装vscode中插件搜索框输入**js-self-methods**
2. 安装完毕后按**F1**(Win:ctrl+shift+p、Mac:command+shift+p)
3. 输入"**method**"可以查看方法列表.
4. 或者**输入关键词**, 比如**"防抖"**.
